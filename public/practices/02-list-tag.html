List Tag

ภาษา HTML จะมี tag สำหรับการแสดงผลที่เป็นรายการ(list) อยู่ 3 ประเภท คือ

      รายการแบบใช้สัญลักษณ์
      รายการแบบใช้ตัวเลข
      รายการแบบหัวข้อและเนื้อหา

รายการแบบใช้สัญลักษณ์

จะใช้ TAG ชื่อ ul เป็น TAG หลัก เเละเราจะใส่ TAG ที่ชื่อว่า li ข้างใน TAG ul อีกที่
โดย TAG li 1 TAG จะใช้สำหรับ 1 รายการ ถ้ามี 3 รายการก็หมายความว่าจะมี TAG li อยู่ใน TAG ul 3 TAG

ตัวอย่างเช่น

ภาควิชาเทคโนโลยีมีเดีย

  • ดิจิทัลมีเดียเทคโนโลยี
  • พัฒนาเกม
  • ชีวการเเพทย์

รายการแบบตัวเลข

จะมีรูปแบบคล้ายกับรายการแบบสัญลักษณ์โดยเราจะเปลี่ยน TAG หลังจาก ul เป็น ol แค่นั้น ส่วนภายใน TAG ol ก็จะ มี TAG li เป็นตัวกำหนดรายการตามเดิม

ตัวอย่างเช่น

เหตุผลหลักที่ทำให้การเรียนเขียนโปรแกรมเป็นไปอย่างยากลำบาก

  1. อ่านภาษาอังกฤษไม่ออก
  2. อ่านคำศัพท์ไม่ได้
  3. ไม่จดคำศัพท์เฉพาะ

รายการแบบหัวข้อและเนื้อหา

รูปแบบนี้จะแตกต่างจากสองแบบแรกอยู่พอสมควรคือ TAG หลักคือ dl แต่ภายในจะไม่ใช่ TAG li
โดยจะเปลี่ยนไปใช้ 2 TAG ด้วยกันคือ dt ใช้สำหรับรายการที่เป็นหัวข้อ และ dd ใช้สำหรับรายการที่เป็นเนื้อหา
และเเน่นอนว่า ทั้ง 2 TAG นี้จะต้องอยู่ใน TAG dl

List ซ้อน List

ในบางครั้งเราสามารถมี List ย่อยอยู่ใน List ใหญ่ได้ โดยวิธีการคือ ให้ทำ TAG หลักของ List (ul,li,dl) ไปใส่ไว้ในไว้ใน TAG li จากนั้นก็ใส่ TAG li ใน TAG หลัก
ที่พึ่งเพิ่มเข้าไป ตามปกติ
