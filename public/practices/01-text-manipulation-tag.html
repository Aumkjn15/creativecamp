เพราะว่า Web Browser จะทำการมองการช่องว่างต่างๆ เป็นแค่การเว้นวรรคหนึ่งครั้งเท่านั้นทดสอบได้ด้วยการเว้นวรรคยาวๆ ในไฟล์ HTML

ข้อความต่างๆ ก็สามารถอยู่ได้โดยไม่มี TAG ครอบเช่นกัน แต่เราควรจะใช้ประโยชน์จาก TAG ต่างๆ มากกว่า
